# 求职/转型反诈 Rubric —— "这岗位/培训是真的吗"判定标准

> 转型期的人最焦虑、信息差最大，是骗局重灾区。任何岗位、培训、内推机会在被推荐给用户前，必须过一遍本文档。
> 原则：**可信度靠机制，不靠话术好听。** 一个把前景吹得天花乱坠的机会，命中红线照样淘汰。

---

## 第一性原理（先记住这个）

真实的招聘，钱的流向永远是**单向的：公司付钱买你的劳动**。

> 凡是反过来要**你先掏钱**，或把"入职/学习"和"贷款/付费"绑在一起的，本质都不是招聘，是收割焦虑。

判定时先问一句：**"这一步要不要我先付钱 / 先承担债务 / 先交个人证件？"** 只要是，就直接拉红线。

---

## A. 硬红线（命中任意一条 = 高危 ❌，直接淘汰）

| # | 红线 | 典型形态 |
|---|---|---|
| A1 | **培训贷 / 先学后付陷阱** | "免费学、包就业、工资里扣"，实为诱导你签消费分期贷款；学不出/没就业，贷款照还 |
| A2 | **付费内推 / 买岗位** | 交钱"保offer""内推保过""买实习证明"——正规公司招聘永远不向候选人收费 |
| A3 | **包就业 / 收益承诺** | "学完月薪过万""保底 X 万""稳进大厂"——就业由市场决定，没人能担保 |
| A4 | **入职前收费** | 押金/服装费/体检指定费/工牌费/培训费/买设备激活才能上岗 |
| A5 | **资金/账户/证件经手** | 让你刷单垫付、过账、出借身份证·银行卡·手机卡、用你名义注册公司（**涉嫌违法犯罪**） |
| A6 | **挂羊头招聘** | 名为"AI运营/助理"，实为色情擦边、传销、电诈引流、灰产引流；面试就拉群发"任务" |

> 命中 A5/A6 的，除了淘汰，**明确提醒可能违法甚至涉刑**，劝退别含糊。
> A1 培训贷是转型人群头号坑：把"学习机会"包装成"零门槛"，代价藏在贷款合同里。见到"分期""扣工资抵学费"立刻警惕。

---

## B. 存疑信号（命中需额外验证，不直接淘汰，但必须如实告知）

| # | 信号 | 为什么可疑 |
|---|---|---|
| B1 | JD 高薪但要求含糊、什么人都要 | "无需经验、日结高薪、会打字即可"——正经高薪岗不会零门槛 |
| B2 | 公司查无此人 / 注册地与办公地对不上 | 企查查搜不到、刚注册、无社保参保人数 |
| B3 | 面试只在微信/QQ、不到正规场所、急着拉群 | 绕开正规流程，跑路成本为零 |
| B4 | 催"名额有限""今天不报明天没了" | 制造稀缺逼你冲动签约/付费 |
| B5 | 培训机构承诺与就业强绑定、晒"高薪学员"无从核实 | 晒的 offer 截图常是话术，无第三方可查 |
| B6 | 薪资远高于同岗行情 | 同岗位别人给的数倍 → 多是钓饵或有隐藏代价 |

---

## C. 验证动作（对"存疑"或用户想确认的机会，实际去查，别凭记忆）

用 WebSearch / WebFetch 实时查：
1. **搜负面**：`<公司/机构名> 骗局` / `<名称> 培训贷` / `<名称> 投诉 黑猫` / `<名称> 靠谱吗`
2. **查实体**：企查查/天眼查搜公司是否真实存在、注册时长、有无社保参保、有无司法风险
3. **查口碑**：脉脉/小红书/知乎/贴吧搜真实在职或受骗反馈，区分"软文/水军"和"真人吐槽"
4. **验证路径**：正规岗位应能走官方招聘渠道核对；培训应能拿到**不绑贷款**的付费方式
5. **薪资行情核对**：用 BOSS 同岗位量级 + 行情报告核对，对方报价是否离谱

---

## C′. 时效核查（强制）

岗位需求、薪资行情、培训内容变化极快。一条过时信息会让人按已失效的玩法准备，等于白费几个月。
1. 查每条关键来源（岗位趋势/薪资/技能要求）的发布日期。
2. 12 个月内可用；12–24 个月标"可能过时"并交叉验证；超 24 个月默认失效。
3. 检索加年份（`2026`/`最新`），优先近期结果。
4. 呈现时每条关键数字标来源年份，整体注明"基于 <今天日期> 检索"。

---

## D. 判定输出格式
```
机会：<岗位/培训/内推名称>
钱的流向：<公司付你 / 还是要你先付？>
硬红线：<命中的 A 编号，或"无">
存疑信号：<命中的 B 编号，或"无">
信息时效：<关键来源年份；是否交叉验证；过时风险>
判定：高危 ❌ / 存疑 ⚠️ / 可行 ✅
验证第一步：<今天就能做、能确认真伪的动作，如企查查搜公司 / 走官方渠道核对>
诚实提示：<对转型人群最该警惕的那一点>
```
判定规则：命中任一 A → 高危；命中 B 且未查清 → 存疑；无 A、B 查清、钱单向流向你 → 可行。

---

## E. 期望管理（每次推荐都带的清醒剂）
- 正规招聘**永远不向你收费**。任何"入职/学习要先掏钱或背贷款"的，先假定是坑。
- "零门槛、包就业、快速高薪"三个词同时出现，骗局概率极高——真正的高薪岗都有真实门槛。
- AI 不会替你上岸，AI 只是让你学得更快、作品出得更多；能不能拿 offer 仍取决于真实能力和市场需求。
- 护城河来自你真学到的能力和作品，不来自任何机构的"包过承诺"。
